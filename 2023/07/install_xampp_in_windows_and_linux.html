    在Linux和Windows环境下,安装和启动xampp以及公网IP访问PHPMyAdmin的流程

    1. 前言

    上本科课程 <基于Linux的web开发>时, 用到了软件 xampp, 记录一下安装和启动xampp以及公网IP访问PHPMyAdmin的流程

    2. 下载xampp

    下载地址:

    Linux: https://sourceforge.net/projects/xampp/files/XAMPP%20Linux/8.2.4/xampp-linux-x64-8.2.4-0-installer.run/download

    # 命令行下载
wget https://sourceforge.net/projects/xampp/files/XAMPP%20Linux/8.2.4/xampp-linux-x64-8.2.4-0-installer.run/download

    Windows: https://sourceforge.net/projects/xampp/files/latest/download

    如果下载链接失效也可以访问官网进行下载:

    点击访问xampp官网的下载页面, Windows和Linux的下载地址如下图:

    3. 安装和启动xampp

    Windows下十分简单, 可以看一下这篇Windows版本的xampp的教程

    我简要介绍一下在Linux下安装和启动xampp, 参考了这篇文章

    # 添加可执行权限
chmod 755 xampp-linux-x64-8.2.4-0-installer.run 
# 安装
./xampp-linux-x64-8.2.4-0-installer.run
# 启动：
sudo /opt/lampp/lampp start
# 停止：
sudo /opt/lampp/lampp stop
# 重启：
sudo /opt/lampp/lampp restart
    • 通过图形化界面管理MySQL数据库

    启动xampp后, 打开浏览器, 输入http://localhost/phpmyadmin/

    4. 通过公网IP访问xampp管理界面

    警告: 这只是我个人的方法, 不一定安全

    1. 打开文件:
    vi /opt/lampp/etc/httpd.conf

    找到下面这一行, 将其下面的内容修改为 Require all granted

          # Controls who can get stuff from this server.
      #
      Require all granted # 修改后的样子
    1. 打开文件: (这一步参考了XAMPP FAQs for Linux)
    vi /opt/lampp/etc/extra/httpd-xampp.conf

    找到下面的内容: 

    Alias /phpmyadmin "/opt/lampp/phpmyadmin/"
<Directory "/opt/lampp/phpmyadmin">
AllowOverride AuthConfig
Require local

    替换为: 

    Alias /phpmyadmin "/opt/lampp/phpmyadmin/"
<Directory "/opt/lampp/phpMyAdmin">
AllowOverride AuthConfig
Require all granted
    1. 下面这一步不一定需要

    根据网上的教程, 在进行前两步后应该可以通过公网IP访问了, 但是我在进行前两步后仍然不能通过公网IP访问, 我在浏览器输入 你的公网IP/dashboard后无反应. 之后我又进行了如下的尝试:

    • 设备信息: 我使用的是阿里云服务器 ECS, 系统为 ubuntu_22_04_x64_20G_alibase_20230208.vhd
    • 我首先按照这个教程: Ubuntu系列弹性云服务器如何安装图形化界面？安装了ubuntu的图形化界面
    • 然后安装Chrome
    • 再在云服务器的图形化界面上打开Chrome, 访问 localhost (这一步是为了检查xampp是否能在本地访问), 成功访问
    • 神奇的是, 这个时候再在本地浏览器上输入 你的公网IP/dashboard 后, 也可以成功访问了. 我不清楚其中的原理, 总之能访问就行了
