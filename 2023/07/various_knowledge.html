    epigenomics and epigenetics

    Epigenetics and epigenomics are related but different fields of study. Epigenetics studies the processes that regulate gene expression, such as DNA methylation, histone modifications, and non-coding RNAs. Epigenomics studies the effects of these epigenetic changes across the entire genome, using techniques such as DNA methylation arrays, chromatin immunoprecipitation, and bisulfite sequencing.

    我的理解: epigenetics 研究调控基因表达的 过程, 而 epigenomics 研究由 epigenetics 的产生的 影响

    4. 宏基因组

    5. 测序深度（Sequencing Depth）

    测序深度（Sequencing Depth）：测序得到的碱基总量（bp）与基因组大小（Genome）的比值，它是评价测序量的指标之一。

    测序深度与基因组覆盖度之间是一个正相关的关系，测序带来的错误率或假阳性结果会随着测序深度的提升而下降。重测序的个体，如果采用的是双末端或Mate-Pair方案，当测序深度在10~15X以上时，基因组覆盖度和测序错误率控制均得以保证。

    6. 共识聚类

    共识聚类是一种聚合（可能相互冲突的）来自多个聚类算法的结果的方法。也称为聚类集成或聚类（或分区）集成，它描述了对于给定数据集已获得许多不同（输入）聚类并且希望找到单个（共识）聚类的情况。观点。在某些方面，它比现有的聚类更适合。因此，共识聚类是协调来自不同来源或同一算法不同运行的同一数据集的聚类信息的问题。作为一个优化问题，共识聚类被称为中值分裂，并且已被证明是 NP 完全的，即使输入聚类的数量为三个。无监督学习中的共识聚类类似于监督学习中的集成学习。

    来源: 共识聚类 Consensus Clustering 最新的百科全书、新闻、评论和研究

    7. R中的extdata

    Extdata is a convention used by some R packages to store external data files, such as data files used for examples in the package documentation or vignettes. It is not a formally defined term, but a subdirectory of the inst directory that gets copied to the installation directory of the package. To access the files in the extdata directory, you can use the system.file function with the package name and the file name as arguments¹³⁴. For example:

    fpath <- system.file(“extdata”, “my_raw_data.csv”, package=”my_package”)

    This will give you the absolute path to the file on your hard drive. You can then use read.csv or other functions to read the file.

    8. 表观转录组学

    表观转录组学（Epitranscriptomics）是近来兴起的热门领域之一，主要研究RNA所携带的化学修饰对基因表达的影响。生物体内的很多大分子都存在化学修饰，且都具有极其重要的作用。比如DNA或组蛋白上的化学修饰参与介导表观遗传调控，许多蛋白（尤其是酶类蛋白）的活性受到多种化学修饰的调节等等。迄今为止，在RNA上已发现了一百多种化学修饰。这些修饰大量分布在非编码RNA（ncRNA），特别是rRNA, tRNA和snRNA上，为ncRNA在翻译与剪接中发挥正常功能所必需。令人兴奋的是，研究人员发现m6A（N6-methyladenosine），m1A（N1-methyladenosine），m5C（5-methylcytidine），hm5C（5-hydroxylmethylcytidine），I（inosine）以及ψ（pseudouridine）等化学修饰也分布在真核生物mRNA上，影响mRNA的代谢与功能。特别是伴随着许多mRNA修饰酶（Writer）、去修饰酶（Eraser）和修饰识别蛋白（Reader）的新发现，mRNA化学修饰的可逆变化与动态调控重新激起了研究人员的兴趣，受到越来越多的关注

    来源: 表观转录组学-康成生物丨数谱生物
    2023-0727

    9. 基因功能元件

    基因功能元件是指控制基因表达的调控元件，大致上可以分成三类：启动子、增强子、边界元件。[1] 启动子是指在哺乳动物中，核心启动子的定义是一段最短的、能够准确启动基因表达的连续DNA序列，通常在转录起始位点（TSS）上下游30~40bp范围，这个区域可以作为转录前起始复合体结合的位置，并且整合刺激和抑制转录的信号（如RNA聚合酶Ⅱ的招募和转录起始等）。[2] 增强子一般认为是一段能够激活/增强基因转录活性的DNA序列，并且其功能与其位置、距靶基因的距离和方向都没有关系，增强子上通常也会有TF的结合位点。[1] 边界元件最早是在黑腹果蝇中鉴定的，又被称为绝缘子，能够间隔常染色质和异染色质，防止他们之间的互相扩散，同时也能阻隔enhancer对promoter发挥作用。[1]

    12. Genomic Feature

    This is a generic and general term to describe any genomic region with some annotated function. E.g. a gene, CDS, mRNA, rRNA, tRNA, repeat sequence, inverted repeat, miRNA, siRNA, origin of replication. A given genomic feature may be comprised of multiple specific genomic locations that together form the genomic feature. For example, a protein coding gene (CDS) is encoded by a gene that has multiple exons. Each exon is spliced together to form a mature mRNA. The CDS feature itself is therefore comprised of multiple genomic locations that neighbor one another.

    来源:
    Genomic Feature - CoGepedia

    13. Functional annotation

    Functional annotation is the process of attaching biological information to sequences of genes or proteins. The basic level of annotation is using sequence alignment tool BLAST for finding similarities, and then annotating genes or proteins based on that. Nevertheless, nowadays more and more additional information of biological functions is added to the annotation system. The additional information allows hand-operated annotation to distinguish genes or proteins that have the same annotation. With many genomes sequenced, computational annotation approaches to characterize genes and proteins from their sequence are increasingly important.
