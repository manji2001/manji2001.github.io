    实现鼠标悬停在文字上时,改变文字的颜色和背景的颜色

    技巧 css 实用技巧
    发布时间 : 2023-07-13 16:17
    字数:894 阅读 :
    1. 1. 前言
    2. 2. 实现步骤
    3. 3. 在vscode中便捷地实现这一效果
    4. 4. 测试文本
    5. 5. 补充

    1. 前言

    在逛 bangumi时, 发现了一个有趣的事情. 句子默认以黑色块的形式展现(比如这样: 巴拉巴拉…), 当鼠标悬停在句子上后, 浮现出白色的文字, 挺有意思, 我下面将在 markdown 中实现这一效果.虽然说markdown的优点之一是: “排版语法简洁，让人们更多地关注内容本身而非排版”, 这么做显得有些花里胡哨了(～￣▽￣)～

    2. 实现步骤

    这其实就是一个简单的css改变文字显示效果的应用.

    1. 首先在文章的任意位置加上下面的代码(我比较喜欢加在文章的顶部位置)
    <style>
.text_mask
{
    font-family: 'SF Pro SC','SF Pro Display','PingFang SC','Lucida Grande','Helvetica Neue',Helvetica,Arial,Verdana,sans-serif,'Hiragino Sans GB';
    word-wrap: break-word;
    line-height: 1.6;
    font-size: 14px;
    white-space: normal;
    border-radius: 2px;
    background-clip: padding-box;
    -webkit-transition: all .5s linear;
    text-shadow: #555 0 0 20px;
    background-color: #555;
    color: #555;
    border: 1px solid #555;
}
mask:hover{
color:white;
background-color:#555;
}
</style>
    1. 然后将想要想要隐藏的句子填写在下面代码的 $1 处就完成了
    <mask class="text_mask">$1</mask>

    3. 在vscode中便捷地实现这一效果

    在 visual studio code(vscode) 软件中, 我们可以自定义代码片段. 通过代码快速补全, 我们可以快速地完成上面的代码. 具体步骤如下:

    1. 点击vscode走下方的 齿轮 按钮,
    2. 点击 User Snippets, (大概在倒数第三项)
    3. 在弹出的框中选择 markdown.json
    4. 复制下列代码, 粘贴到 markdown.json 文件中
        "text-mask-css":{
        "prefix": "text-mask-css",
        "body": [
            "<style>\n.text_mask\n{\n    font-family: 'SF Pro SC','SF Pro Display','PingFang SC','Lucida Grande','Helvetica Neue',Helvetica,Arial,Verdana,sans-serif,'Hiragino Sans GB';\n    word-wrap: break-word;\n    line-height: 1.6;\n    font-size: 14px;\n    white-space: normal;\n    border-radius: 2px;\n    background-clip: padding-box;\n    -webkit-transition: all .5s linear;\n    text-shadow: #555 0 0 20px;\n    background-color: #555;\n    color: #555;\n    border: 1px solid #555;\n}\nmask:hover{\ncolor:white;\nbackground-color:#555;\n}\n</style>",
        ]
    },
    "text-mask":{
        "prefix": "text-mask",
        "body": [
            "<mask class=\"text_mask\">$1</mask>",
        ]
    },

    现在我们在 markdown 文件中输入 text, 就会出现如下的选项:

    我们可以通过选择 text-mask-css 或 text-mask 进行快速补全啦~

    4. 测试文本

    读者可以将下面的文本粘贴到 markdown 文件中, 测试效果

    5. 补充

    上面的代码中, 最核心的是下面这一部分, 其他的代码是为了设置字体的大小, 字体的阴影效果等

    <style>
.text_mask
{
    background-color: #555;
    color: #555;
}
mask:hover{
color:white;
background-color:#555;
}
</style>
